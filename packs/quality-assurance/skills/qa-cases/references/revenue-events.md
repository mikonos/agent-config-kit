# Revenue / Operations Event Test Add-On

Use when the feature is a campaign, event page, lottery, reward, ranking, CP/social binding, announcement, popup, or cost-controlled operation.

## Coverage

- Activity time.
- Special daily windows.
- Binding / relationship states.
- Tickets / points / characters.
- Lottery pool.
- Reward inventory.
- High-value reward CD.
- Personal limits.
- Downgrade.
- Ranking score.
- Daily/total settlement.
- Blacklist.
- Announcement.
- Popup.
- Activity record.
- Cost push.
- Resource table.

## Required Test Hooks

- Time simulation.
- Forced reward hit.
- Forced success/failure.
- Inventory query.
- Reward issuance log.
- Ranking raw score query.
- User state setup.
- Blacklist setup.
- Broadcast trigger log.
