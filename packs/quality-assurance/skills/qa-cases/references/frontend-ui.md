# Frontend / UI Test Add-On

Use when the requirement is a page, modal, form, dashboard, editor, mobile screen, or frontend-heavy feature.

## Coverage

- Initial loading.
- Empty state.
- Error state.
- Success state.
- Disabled state.
- Form validation.
- Long text.
- Special characters.
- Responsive layout.
- Scroll behavior.
- Sticky/fixed elements.
- Modal open/close.
- Back navigation.
- Refresh.
- Multi-tab behavior.
- Permissions in UI.
- Accessibility basics.
- Analytics events if required.

页面入口
页面展示

## Visual Checks

- No overlapping text.
- No clipped button labels.
- Loading skeleton does not shift layout badly.
- Empty and error states have actionable copy.
- Disabled controls cannot still submit.
- Toasts do not stack excessively.
