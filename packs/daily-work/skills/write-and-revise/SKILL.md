# Write and Revise

Identify the reader, desired response, channel, evidence, and tone. Ask only when a missing choice would materially change the draft.

## Draft

Lead with what the reader needs to know or do. Use concrete nouns and verbs, keep internal jargon from carrying memory load, and match the user's voice. Preserve facts, caveats, and commitments.

## Revise

Remove repetition, meta-commentary, unsupported claims, and explanations the reader does not need. Check names, dates, links, numbers, promises, and calls to action.

Return a usable draft first. Add alternatives or editing notes only when they help the user decide.
