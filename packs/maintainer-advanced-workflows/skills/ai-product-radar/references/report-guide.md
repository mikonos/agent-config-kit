## 1. 决策优先：报告开头三件事

原版开头先堆 8 条信号，读者要翻很久才知道"那我该干嘛"。本版强制**倒金字塔**：

1. **一句话裁决**：本轮 N 个机会 → X 个布局 / Y 个观望 / Z 个放弃。
2. **单一首推赌注**：哪一个、机会分多少、为什么是它、最快多少天能证伪。
3. **雷达图**（右上黄金区一眼可见），再往下才是信号与评分明细。

## 2. 溯源纪律（贯穿全文）

- 每个市场规模 / 竞品收入 / CAGR：贴链接，或标 `（推测）`。
- 竞品收入用区间（"<$5K MRR"）而非假精确（"$4,237"）。
- 严禁把 `speculative` 数字写成事实陈述句。一致性自检见 scoring-rubric.md §5。

## 3. 深档七要素怎么写（每个 Top 机会一份）

| 要素 | 写到位的标准 | 原版对应 |
|---|---|---|
| **痛点 + 用户画像** | 一句话痛点 + 一个具体的人（年龄/身份/已用的 workaround） | §3.1 |
| **竞品现状表** | 列 4–6 个竞品：收入区间 / 评分 / **共同缺口**（差异化楔子从这来） | §3.2 |
| **AI 差异化** | 用"传统做法 → AI 做法"对比框，点明 AI 是核心解锁还是装饰 | §3.3 |
| **MVP 路径** | 按天拆（3–7 天），每天一个可交付 + 用什么工具 | §3.4 |
| **新手引导 5 屏** | 情感共鸣→建立信任→低摩擦上手→Aha→付费墙；记住"引导占成败 70%" | §3.5 |
| **预估指标** | 保守/乐观双列：TAM、下载、转化、MRR、达 $5K 时间、CAC、LTV | §3.7 |
| **Go/No-Go** | 见下节 | §3.8 |

## 4. 三层验证 + Go/No-Go + 退出标准

**核心心法：内容验证先于产品构建。先确认有人看 → 再确认有人付费 → 最后才构建。**

| 层 | 内容 | 周期 | 成本 | Go 标准 |
|---|---|---|---|---|
| Layer 1 | 匿名内容账号（TikTok/小红书）测需求 | 3–5 天 | $0 | ≥1 条 >1 万播放 + 10+ 需求留言 |
| Layer 2 | 社区/DM 验付费意愿 | 1–2 周 | $0 | >50 条需求 + >10 人求提前体验 |
| Layer 3 | MVP + **强制付费墙** | 30 天 | ~$100 API | ≥10 个付费用户 |

**30 天验证截止**：超时即放弃，不给沉没成本绑架的机会。免费好评 ≠ 付费意愿，必须有人掏真金白银。

## 5. 内置证伪（本版新增 · 每个 Top 机会必写）

原版只讲机会多诱人，不讲它怎么会死。每份深档**必须**含一段"**反面**"：

- **杀死条件**：什么情况发生就立刻停（如：Layer 1 五天内 0 条破万；竞品已在同方向拿到 traction；该痛点其实是小众长尾）。
- **最强反方一句话**：站在唱反调者立场，这个机会最致命的弱点是什么。
- **已知不利证据**：扫描中遇到的、与"这是好机会"相悖的信号（原版 Step 0 审查精神）。

> 没有反面就没有差异；不刻入差异就没有思考。一个没有"杀死条件"的机会，是广告不是分析。

## 6. GTM / ASO / 指标（Phase 6）

- **推广渠道矩阵**：匿名内容账号（上架前 5 天起）/ UGC 创作者（$15 条）/ 网红 CPM 测试 / 创始人内容 / ASO / 博主合作 / 付费广告（$5K MRR 后才开）。每行：成本 / 启动时间 / 预期 / 执行要点。
- **内容一致性 = 品牌**：固定视觉 + 固定 IP 人格 + 爆款结构复制（换文案不换框架）。
- **ASO 检查清单**：名称含关键词 / 副标题 30 字内（决定转化）/ 第一张截图"丑但清晰 > 好看但困惑" / 描述前 3 行最重要 / 付费墙在引导第 5 步不可跳过。
- **指标追踪表**：播放总量、爆款数、需求留言、提前体验请求、MVP 状态、付费用户、MRR、引导→付费墙转化、DAU，按 W1–W4 填。

## 7. 反模式清单（写完自查，命中即改）

- ❌ 开头堆信号、不给裁决 → 违反决策优先
- ❌ 市场数字无出处又写成事实 → 违反溯源纪律
- ❌ 机会分最高的不是首推的，且不解释 → 违反一致性自检
- ❌ 深档全是优点、没有"杀死条件" → 违反内置证伪
- ❌ 雷达数字和报告表格对不上 → 必须由 build_radar.py 统一生成
- ❌ 给一次性建议、不讲"每月重跑" → 违反系统化（机会会随竞品入场失效）
