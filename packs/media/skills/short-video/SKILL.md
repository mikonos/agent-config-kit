# 短视频创意策划

## 核心目标
策划具有吸引力和传播性的短视频创意方案，提升完播率和互动率。

## 快速开始
- 先明确：平台、时长、目标（曝光/认知/转化/互动）、受众与“必须出现的卖点/禁区”。
- 输出优先给 3-5 个方向，再按用户选中的方向细化到分镜/台词/镜头节奏。

## 与其他技能联用
- `rednote-caption-to-screenshots`：当你要把“小红书内容”最终交付为**轮播图/长图截图**（而不仅是脚本/分镜）时，用它完成“配文→排版→截图”的出图闭环。

## 适用场景
- 产品功能展示
- 品牌营销推广
- 电商带货视频
- KOL合作内容

## 基础参数

| 维度 | 选项 |
|------|------|
| 时长 | 10秒/15秒/30秒/60秒 |
| 比例 | 9:16竖屏/16:9横屏/1:1方形 |
| 平台 | 抖音/快手/小红书/B站/YouTube Shorts |
| 目标 | 品牌曝光/功能认知/购买转化/用户互动 |
| 调性 | 搞笑幽默/专业科普/温馨感人/炫酷炸裂 |

---

## 工作流

### Step 1: 核心卖点提炼
- 核心功能特点
- 独特价值主张
- 用户痛点解决
- 竞品差异化优势

### Step 2: 创意方向设定

| 类型 | 说明 |
|------|------|
| 反差对比型 | 普通方式 vs 产品方式的强烈对比 |
| 夸张演绎型 | 将功能进行夸张化、戏剧化展示 |
| 场景植入型 | 将产品自然融入生活场景 |
| 悬念揭秘型 | 设置悬念，逐步揭示功能 |
| 搞笑段子型 | 用幽默方式展示产品特点 |
| 挑战测试型 | 通过各种测试证明效果 |

### Step 3: 内容结构设计
**黄金3秒法则**：
- 0-1秒：强烈视觉冲击，抓住注意力
- 1-3秒：建立悬念或明确主题
- 3-末尾：功能展示和情感满足

## 输出模板

```
你是一个短视频创意专家，请你基于【产品名称】的【核心功能】这个选题，
为我策划【数量】个【风格类型】的【时长】短视频画面，用表格的形式输出。

输出格式：
| 序号 | 创意概念 | 静态描述 | 动态描述 | 预期效果 |
|------|----------|----------|----------|----------|
```

---

## 评估标准

**内容质量**：
- 前3秒能否抓住注意力
- 是否有容易记住的亮点元素
- 是否容易引发分享

**商业价值**：
- 产品功能是否有效展示
- 是否能激发购买欲望

---

## 质量检查
- [ ] 每个创意都明确：Hook（前 1-3 秒）→ 冲突/反差 → 价值呈现 → CTA
- [ ] 创意与目标一致（曝光 vs 转化的结构与文案不同）
- [ ] 画面描述可直接给拍摄/剪辑执行（避免抽象词）
