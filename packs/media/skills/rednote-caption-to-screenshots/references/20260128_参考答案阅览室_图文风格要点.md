# 参考答案阅览室 图文风格要点（可模仿）

> 来源：小红书账号「参考答案阅览室」5 篇图文抓取与归纳。当用户要求「参考答案阅览室风」「深度阅读风」「知识类轮播像参考答案阅览室」时，按本要点写标题与正文。

## 1. 封面/标题

- **句式**：矛盾/痛点作标题，或「人群 + 一定要看/一定要懂」
  - 例：`过度期待，是一种隐形的自我虐待`
  - 例：`又到年底了，找不到意义的人一定要看`
- 避免纯结论句、无情绪钩子。

## 2. 正文结构（短段 + 列表）

- **开篇**：1–2 句场景/共鸣。
- **转折**：一句引入「这篇文章/本期内容/他提出…」
- **主体**：用 **▸** 做 3 档并列，或用 **1️⃣2️⃣3️⃣** 做 3–5 条提炼（每条一句）。
- **收尾**：一句金句或祝福（「当你学会…」「以上，就是本期分享，祝…」）。

## 3. 语言节奏

- 短句为主（单句尽量 ≤25 字）。
- 可有一处反问/自问自答。
- 适当数字化（「三个桶」「五个点」「2% 的恐惧」）。

## 4. 标签与 CTA

- 标签：深度阅读、深度思考 + 主题长尾，约 5–8 个。
- CTA：轻引导（「左滑看原文」「本期分享」），不硬广。

## 5. 与 8 页轮播的对应

- P1 封面：矛盾点或人群+承诺
- P2 痛点/场景：1–2 句共鸣
- P3 结论：一句话「这篇文章/他提出…」
- P4–P5 方法：1️⃣2️⃣3️⃣ 或 ▸▸▸
- P6 金句/祝福
- P7 清单（可选）
- P8 CTA
