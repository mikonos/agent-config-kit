# 缠论分析 Skill · 完整包 —— 先读我 👋

这是一个**即下即用的缠论分析 Skill 包**。把缠中说禅（缠论）的技术分析体系，做成一套能让**任何 AI** 帮你按规则看盘的操作系统——核心是纪律：**先定级别、先认结构、先写失效点**，数据不够就老实降级，绝不自欺。

> ⚠️ **免责声明**：仅供技术研究与学习，**不构成投资建议，不给买卖指令，不承诺收益**。缠论有主观性和失效风险，盈亏自负。它帮你的是"少做错单、看清结构"，不是"抓必涨"。

## 怎么用（两种方式，任选其一）

### 方式 A · 任意 AI（最简单，推荐新手）

适用 DeepSeek / 豆包 / Kimi / 腾讯元宝 / 通义千问 / ChatGPT / Claude 等**任何 AI**：

1. 打开 **`缠论Skill_完整版_通用AI.md`**，**整段复制全文**。
2. 粘贴进 AI 对话框，发送。
3. 再发一句你的需求，例如：
   > "现在你就是上面这套缠论分析师。帮我按这套规则分析【某标的】的【日线】走势：先定级别、再认结构、最后给买卖点判断和失效条件。数据不够就老实降级，别直接给我买卖建议。"
4. 有 K 线截图（最好带 MACD、成交量）就一起发，AI 会读图。

> 没有截图也能用：AI 会给你一套自检清单 + 假设分析，而不是假装看到了图。

### 方式 B · Claude Code（开发者）

1. 使用当前 Agent Runtime 的项目级 Skill 安装方式复制**整个文件夹**。
2. 新开一个会话，问它"用缠论帮我看下 XXXX 的日线走势"，会自动触发。
3. Claude 会按需加载 `references/` 里的细则。

## 它能做什么 / 不能做什么

| 不是 | 是 |
|---|---|
| ❌ 一个"必涨信号"机器 | ✅ 一套强制纪律的分析流程 |
| ❌ 指标金叉就喊买 | ✅ 走完 包含→分型→笔→线段→中枢→背驰→买卖点 才下结论 |
| ❌ 把突破当三买 | ✅ 中枢离开+回拉+不回 全点名才认三买 |
| ❌ 假装看懂任何图 | ✅ 数据不够就降级成"观察" |
| ❌ 替你决定买卖和仓位 | ✅ 告诉你"结构在什么条件下成立 / 失效" |
