# 从这里开始 / Start here

你不需要记住 Skill 名称。直接用普通话说出想得到的结果；如果不知道怎么开始，就说：

```text
请帮我开始处理这件事：……
```

Agent 会在阅读、循证研究、计划执行、写作和验收中选择最小够用的流程。
项目原有的、更具体的规则仍然优先。

You do not need to remember skill names. Tell the Agent the outcome you want in
ordinary language. If you are unsure how to begin, say:

```text
Help me start this task: ...
```

The `start-here` skill selects the smallest useful workflow. The Agent should
ask only for information that changes the result, permission, or an irreversible action.
